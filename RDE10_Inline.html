Cascading Style Sheets - Folhas de Estilo em Cascata

Tem por objetivo realizar a estilização de conteúdo de documentos HTML. Podem definir cores, fontes, entre outros.

Sintaxe:

  • Seletor: seletor de elementos/tags para buscar no documento html.
  • "Propriedades:valor": valor desejado para alterar na propriedade.

Como exemplo, o seletor do elemento "p", onde ele irá procurar todos os "p" que estiverem no documento html para fazer um tipo de estilização, como no exemplo abaixo, irá mudar a cor para azul.

FORMAS DE UTILIZAR O CSS

São três:

Inline:
Os código CSS são definidos dentro de uma tag HTML.
Afetam somente a tag em que foi definida.

Algumas sintaxes utilizando o inline:

  • style="background-color: - usa-se para colorir todo o fundo do site quando colocado dentro da tag body.
  • style="color: - define a cor da fonte.
  • background-color: - define a cor de fundo de algum título ou alguma outra escrita.
  • text-align: center - utilizado para alinhar o conteúdo.
  • style="font-size: - define o tamanho da fonte.
Incorporado:
Os código CSS são definidos dentro do cabeçalho, dentro da tag head.
Afetam somente o documento em que foi definido.
Externo:
Os código CSS são definidos em um arquivo externo.
Afetam todos os documentos que estão linkados ao CSS.
