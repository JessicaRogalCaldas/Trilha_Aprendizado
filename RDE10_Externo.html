CSS - Cascading Style Sheets

EXTERNO:

Os código CSS são definidos em um arquivo externo.
Afetam todos os documentos que estão linkados ao CSS.
A vatagem é que aestilização é feita a parte, terá assim o arquivo de conteúdo Html e o código CSS em arquivos distintos.
O Externo pode ser aplicado à várias páginas.

Entre a tag head será feito um link onde será dito que será uma folha de estilo que será linkada. Para criar a pasta do ccs e o aqruivo, basta apertar o ctrl e clicar em cima. Aparecerá uma mensagem, então será possível clicar na opção criar arquivo.
Desta forma a pasta e o arquivo css já estará no diretório.

Feito isso, no arquivo do css criado começa a estilização.
